# NetSwitcher
Switch WLAN/Ethernet for web/local. Implemented on python.

## Pack py files into exe
```
1. pip install pyinstaller
2. pyinstaller -F -w main.py -i duck_icon.ico
3. find your generated .exe file in dist/.
```
